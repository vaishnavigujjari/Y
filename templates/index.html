Team selection rules

Batsman

User's team must cantain atleast "4" Batsmen.

All-Rounder

User's team must cantain atleast "1" All-Rounder.

Wicket-Keeper

User's team must cantain atleast "1" Wicket-Keeper.


Points System

Points for Runs

User will be awarded "1" point for each run if the player is included in the user's team.

Points for Catch

User will be awarded "10" point for each run if the player is included in the user's team.

Points for Wickets

User will be awarded "20" point for each wicket if the player is included in the user's team.

Points for Captain

User will be awarded "Double" point for each run if the player is included in the user's team.
